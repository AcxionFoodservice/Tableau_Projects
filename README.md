## Generating a report

Reports can be run on demand via GitHub Actions:

1. Go to the **Actions** tab
2. Select **Generate Tableau Usage Report**
3. Click **Run workflow**
4. Download the `.xlsx` from the **Artifacts** section when complete

See each folder's README for full setup and local-run instructions.
